    Create Order (with or without estimate)

    HTTP Request

    POST http://api.gateway.com/order

    Request body

    {
  "price": 123
}

    Response

    {"test":"test"}

    Get Order

    HTTP Request

    POST http://api.gateway.com/order/<id>

    URL Parameters

    Parameter Description
    id Order id 
      {
    "id": 123,
    "price": 123
  }

    Update Order

    HTTP Request

    PUT http://api.gateway.com/order/<id>

    Request body

    {
  "price": 123
}

    Response

    {
  "id": 123,
  "price": 123
}

    Cancel Order

    HTTP Request

    PUT http://api.gateway.com/order/<id>

    URL Parameters

    Parameter Description
    id Order id

    Request body

    {
  "status": "CANCELED"
}

    Response

    {
  "id": 123,
  "price": 123,
  "status": "CANCELED"
}

    Get Order Status

    HTTP Request

    PUT http://api.gateway.com/order/<id>/status

    URL Parameters

    Parameter Description
    id Order id

    Response

      {
    "status": "CANCELED",
    "updated_at": 123123123
  }
